## ➤ Installation

Launch _Quick Open_ depending on your operating system:

- 🐧 **Linux**: `Ctrl+P`
- 🍎 **macOS**: `⌘P`
- 🪟 **Windows**: `Ctrl+P`

Paste the following command and press `Enter`:

```bash
ext install bastndev.lynx-theme
```
