## ➤ Instalación

Inicia _Quick Open_ según tu sistema operativo:

- 🐧 **Linux**: `Ctrl+P`
- 🍎 **macOS**: `⌘P`
- 🪟 **Windows**: `Ctrl+P`

Pega el siguiente comando y pulsa `Enter`:

```bash
ext install bastndev.lynx-theme
```
