## ➤ Instalação

Inicie o _Quick Open_ dependendo do seu sistema operacional:

- 🐧 **Linux**: `Ctrl+P`
- 🍎 **macOS**: `⌘P`
- 🪟 **Windows**: `Ctrl+P`

Cole o seguinte comando e pressione `Enter`:

```bash
ext install bastndev.lynx-theme
```
