## ➤ インストール

オペレーティングシステムに応じて _Quick Open_ を起動します：

- 🐧 **Linux**: `Ctrl+P`
- 🍎 **macOS**: `⌘P`
- 🪟 **Windows**: `Ctrl+P`

次のコマンドを貼り付けて `Enter` を押します：

```bash
ext install bastndev.lynx-theme
```
