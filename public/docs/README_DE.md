## Installation

Starten Sie _Quick Open_ abhängig von Ihrem Betriebssystem:

- <img src="https://www.kernel.org/theme/images/logos/favicon.png" width=16 height=16/> Linux `Ctrl+P`
- <img src="https://developer.apple.com/favicon.ico" width=16 height=16/> macOS `⌘P`
- <img src="https://www.microsoft.com/favicon.ico" width=16 height=16/> Windows `Ctrl+P`

Fügen Sie den folgenden Befehl ein und drücken Sie `Enter`:

```bash
ext install bastndev.lynx-theme
```

### Methode 2 — Erweiterungsansicht

1. Öffnen Sie Extensions (`Ctrl+Shift+X` / `⌘+Shift+X`)
2. Suchen Sie nach **"Lynx Theme Pro"**
3. Klicken Sie auf **Install**
