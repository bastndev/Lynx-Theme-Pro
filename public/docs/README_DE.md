## ➤ Installation

Starten Sie _Quick Open_ je nach Ihrem Betriebssystem:

- 🐧 **Linux**: `Strg+P`
- 🍎 **macOS**: `⌘P`
- 🪟 **Windows**: `Strg+P`

Fügen Sie den folgenden Befehl ein und drücken Sie `Enter`:

```bash
ext install bastndev.lynx-theme
```
