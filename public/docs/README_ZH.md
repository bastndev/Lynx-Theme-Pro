## ➤ 安装

根据您的操作系统启动 _Quick Open_：

- 🐧 **Linux**: `Ctrl+P`
- 🍎 **macOS**: `⌘P`
- 🪟 **Windows**: `Ctrl+P`

粘贴以下命令并按 `Enter`：

```bash
ext install bastndev.lynx-theme
```
