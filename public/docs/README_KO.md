## ➤ 설치

운영 체제에 따라 _Quick Open_ 을 실행합니다:

- 🐧 **Linux**: `Ctrl+P`
- 🍎 **macOS**: `⌘P`
- 🪟 **Windows**: `Ctrl+P`

다음 명령어를 붙여넣고 `Enter` 를 누르세요:

```bash
ext install bastndev.lynx-theme
```
